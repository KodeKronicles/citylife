Responsiveness

The layout of the website has been developed to accommodate different screen sizes, in order to ensure a consistent browsing experience for users on computers and mobile devices. This has been achieved using:

  • CSS media queries: used to customise styling based on device characteristics (in our case: resolution).
  • Flex grid system: Bootstrap's flex grid systems allows for easy adaptation of content display according to used resolution.
  • Scalable components Some components used adjust dynamically to the viewport size, with margins and paddings scaling appropriately.
  • Responsive images: Images are resized according to the resolution used.
